# akvstore
Authenticated key/value store built on redis

AKVStore provides a basic key-value store with super light authentication.

## Configuration
Set the following environment variables
* AKVS_ROOT_APP_ID
* AKVS_API_PORT
* AKVS_API_BASE_URI
* AKVS_PASSWORD_SALT
* REDIS_HOST
* REDIS_PORT
* REDIS_PW
